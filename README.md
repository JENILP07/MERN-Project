## ✨ Features

- ⚛️ **React Frontend** — Component-based UI with hooks and state management
- 🚀 **Express Backend** — RESTful API with Node.js
- 🍃 **MongoDB** — NoSQL database for flexible data storage
- 🔗 **Full-Stack Integration** — Seamless frontend-backend communication
- 📱 **Responsive Design** — Works across devices

## 🚀 Getting Started

### Prerequisites
- Node.js >= 16
- MongoDB (local or Atlas)

### Installation

```bash
# Clone the repository
git clone https://github.com/JENILP07/MERN-Project.git
cd MERN-Project
```

### Backend Setup

```bash
cd Backend
npm install

# Create .env file
echo "MONGO_URI=mongodb://localhost:27017/merndb
PORT=5000" > .env

# Start backend server
npm start
```

### Frontend Setup

```bash
cd Frontend
npm install

# Start React development server
npm start
```

The app will be available at `http://localhost:3000`.
